Open-source toolkit for building AI-powered companion applications with empathy-driven design.

## Quick Start

```bash
git clone https://github.com/Tricia-42/companion-kit.git
cd companion-kit
npm install
npm run dev
```

Visit http://localhost:8005 to see the Trip demo app.
